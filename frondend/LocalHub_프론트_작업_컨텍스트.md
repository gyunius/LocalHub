# LocalHub 프론트엔드 작업 컨텍스트

## 한 줄 요약
공공데이터 기반 **서울 지역정보 커뮤니티** 웹앱. 로그인 없는 **익명 게시판** + **관광지 지도/코스 공유** + **AI 관광 챗봇**을 3일(Day1 기획·설계 / Day2 개발 / Day3 배포·테스트·발표) 스프린트로 만드는 프로젝트. 프론트는 **Vue.js 3 SPA**.

핵심 사용자 흐름: 관광지 선택 → 방문 순서로 코스 구성 → 익명 게시글로 공유 → 댓글.

## 확정 스택
| 영역 | 기술 |
|---|---|
| 프론트엔드 | Vue.js 3 (SPA) |
| 백엔드 | FastAPI + SQLAlchemy |
| DB | SQLite |
| 챗봇 | OpenAI API (키는 **백엔드에서만** 사용) |
| 배포 | 프론트 = Netlify / 백엔드 = Render |
| 보안 | `.env` + `.gitignore`로 키 분리 |

## 프론트엔드가 만들 화면·컴포넌트
필수(Must)와 팀이 선정한 선택기능(Should/Could)을 프론트 관점으로 정리한 것:

| 구분 | 화면/컴포넌트 | 핵심 포인트 |
|---|---|---|
| Must | 게시글 목록 | pagination(기본형), 조회수 표시 |
| Must | 게시글 상세 | 조회 시 조회수 증가 |
| Must | 게시글 작성 | 제목·내용·수정용 비밀번호 입력 |
| Must | 게시글 수정 | 비밀번호 확인 후 진입 |
| Must | 비밀번호 확인 UI | 수정·삭제 시 비밀번호 일치 검사, 불일치 시 오류 메시지 |
| Must | 플로팅 챗봇 UI | 모바일 대응, 대화 히스토리 유지, `POST /api/chat` 호출 |
| 선택 | 지도 뷰 | 관광지·맛집을 pin + popup으로 표시 |
| 선택 | 코스 빌더 | pin 선택 → 방문 순서 구성 |
| 선택 | 코스 공유 | 만든 코스를 익명 게시글로 등록 |
| 선택 | 댓글 UI | 공유 코스에 댓글 작성·삭제 |
| 선택 | 검색 UI | 제목·내용 keyword 검색 |

보류/제외: 축제 캘린더·좋아요·태그·링크복사 공유는 여유 있으면. 회원가입·로그인, 실제 길찾기, 코스 중개 플랫폼은 이번 범위 밖(Won't).

## 프론트가 호출할 백엔드 API
| 기능 | 엔드포인트 | 비고 |
|---|---|---|
| 게시글 목록/상세/작성/수정/삭제 | REST CRUD (`/api/posts` 계열, 명칭 미확정) | 수정·삭제는 비밀번호 검증 |
| 챗봇 | `POST /api/chat` | 확정. 요청/응답 스키마는 BE와 합의 필요 |
| 코스·댓글·검색·조회수 | 추가 API (미정) | Day2에 BE가 구현 예정 |

기획서에 정확한 엔드포인트·요청/응답 스키마까지는 안 나와 있음. 프론트 작업하다 필요한 시점에 그 자리에서 같이 정의하면 됨.

## 핵심 제약 & 주의점
**인증 없음 / 비밀번호 모델** — 계정·JWT·OAuth 전혀 없음. 글마다 "수정용 비밀번호"로만 권한 제어. 교육 목적이라 평문 저장을 전제로 하지만, 실무 기준으로는 절대 하면 안 되는 방식이라는 것만 알고 가면 됨.

**챗봇 키 노출 금지** — OpenAI 키는 백엔드에서만 쓴다. 프론트에서 직접 OpenAI를 호출하는 순간 키가 브라우저에 노출되니, 반드시 BE 프록시(`/api/chat`)를 경유해야 함.

**배포는 Netlify** — SPA라서 새로고침 시 404 나는 이슈(history mode fallback)를 배포 단계에서 잡아줘야 할 수 있음.

**개발도구 규칙** — 규정상 이 프로젝트는 VSCode Copilot + OpenAI API만 허용이고 Cursor·Claude Code·Codex 같은 코드생성 도구는 금지야. 개념을 묻고 코드를 직접 짜는 건 이 규칙과 별개지만, 내가 통째로 짜준 코드를 그대로 붙여넣는 건 회색지대. 판단은 네 몫이고, 필요하면 "코드 말고 방향·설명만" 식으로 요청해도 돼.

## 작업 방식 (내가 이렇게 이해함)
- 너는 프론트 공부하려고 **질문 위주**로 진행.
- 나는 위 컨텍스트(LocalHub / Vue 3 SPA / 익명게시판 + 지도·코스 + 챗봇 / Netlify)를 매번 전제로 깔고 답함. 배경 다시 설명 안 해도 됨.
- 특정 화면이나 기능(예: "비밀번호 확인 UI", "코스 빌더")을 지목하면 그 맥락에서 바로 이어감.
